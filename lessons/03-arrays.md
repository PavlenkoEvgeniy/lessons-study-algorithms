## Зачем этот урок

Представь шкаф с ящиками:
- В ящике 1 — носки

- В ящике 3 — штаны

Ты знаешь номер ящика и сразу достаёшь нужное. **Массив работает так же** — по номеру (индексу) достаём элемент.

---

## Что такое массив?

**Массив** — это набор элементов, лежащих в памяти рядом друг с другом, где каждый элемент имеет свой номер (индекс).

**Простая аналогия: парковка**

```
Парковка с 5 местами:

Место 0: [Машина A]
Место 1: [Машина B]
Место 2: [Пусто]
Место 3: [Машина C]
Место 4: [Машина D]

Индекс (номер места):  0    1    2    3    4
Элемент:             "A"  "B"  null "C"  "D"
```

## Массив в PHP

### Создание

```php
<?php

// Пустой массив
$empty = [];

// Массив с числами
$numbers = [10, 20, 30, 40, 50];

// Массив со строками
$names = ["Аня", "Борис", "Вика"];

// Смешанный массив
$mixed = [1, "два", 3.0, true];
```

### Доступ по индексу

```php
<?php

$colors = ["красный", "зелёный", "синий", "жёлтый"];

// Индексы начинаются с 0!
echo $colors[0]; // "красный" (первый элемент)
echo $colors[1]; // "зелёный" (второй)
echo $colors[2]; // "синий" (третий)
echo $colors[3]; // "жёлтый" (четвёртый)

// Последний элемент
$lastIndex = count($colors) - 1;
echo $colors[$lastIndex]; // "жёлтый"

// Отрицательный индекс (PHP 7.1+)
echo $colors[-1]; // "жёлтый" — последний
echo $colors[-2]; // "синий" — предпоследний
```

**Важно:** Индексы начинаются с **0**, не с 1!

```
Неправильно думать: 1, 2, 3, 4...
Правильно думать:  0, 1, 2, 3...
```

## Основные операции

### 1. Добавить элемент

```php
<?php

$fruits = ["яблоко", "банан"];

// В конец — быстро O(1) амортизированно
$fruits[] = "апельсин";
print_r($fruits); // ["яблоко", "банан", "апельсин"]

// Или так
array_push($fruits, "груша");
print_r($fruits); // ["яблоко", "банан", "апельсин", "груша"]
```

### 2. Изменить элемент

```php
<?php

$cars = ["BMW", "Audi", "Mercedes"];

// Изменить второй элемент (индекс 1)
$cars[1] = "Volkswagen";
print_r($cars); // ["BMW", "Volkswagen", "Mercedes"]
```

### 3. Удалить элемент

```php
<?php

$items = ["A", "B", "C", "D"];

// Удалить последний
array_pop($items); // ["A", "B", "C"]

// Удалить первый (медленно! O(n) — всё сдвигается)
array_shift($items); // ["B", "C"]

// Удалить по индексу
unset($items[1]); // ["B"] (без переиндексации!)
// unset НЕ сдвигает индексы!

// Если нужно сдвинуть:
$items = array_values($items); // [0 => "B"]
```

### 4. Проверить наличие

```php
<?php

$ages = ["Аня" => 25, "Борис" => 30, "Вика" => 28];

// Проверить значение (линейный поиск O(n))
if (in_array(25, $ages)) {
    echo "Нашли 25!";
}

// Проверить ключ (быстрее O(1) для хеш-таблиц)
if (array_key_exists("Аня", $ages)) {
    echo "Ключ 'Аня' существует";
}

// Получить ключ по значению
$key = array_search(30, $ages);
echo $key; // "Борис"
```

## Сложность операций

```
┌─────────────────────┬──────────────┬─────────────────────────────┐
│ Операция             │ Сложность    │ Пояснение                  │
├─────────────────────┼──────────────┼─────────────────────────────┤
│ Доступ по индексу    │ O(1)         │ Зная адрес, сразу достаём  │
│ Изменение по индексу │ O(1)         │ Тоже мгновенно             │
│ Добавление в конец   │ O(1)*        │ *амортизированно           │
│ Добавление в начало  │ O(n)         │ Надо сдвинуть все элементы │
│ Удаление из конца    │ O(1)         │ Просто убрать последний    │
│ Удаление из начала    │ O(n)         │ Надо сдвинуть все элементы │
│ Поиск элемента       │ O(n)         │ Линейный проход            │
└─────────────────────┴──────────────┴─────────────────────────────┘
```

## Практический пример: Список задач

```php
<?php

/**
 * Простой список задач на массиве
 */

class TodoList
{
    private array $tasks = [];
    
    // Добавить задачу — O(1)
    public function add(string $task): void
    {
        $this->tasks[] = $task;
    }
    
    // Показать все — O(n)
    public function showAll(): void
    {
        foreach ($this->tasks as $i => $task) {
            echo ($i + 1) . ". " . $task . "\n";
        }
    }
    
    // Выполнить задачу (удалить) — O(n)
    public function complete(int $index): bool
    {
        $index--; // Пользователь вводит 1,2,3..., а массив 0,1,2...
        
        if ($index < 0 || $index >= count($this->tasks)) {
            return false;
        }
        
        unset($this->tasks[$index]);
        $this->tasks = array_values($this->tasks); // Переиндексировать
        return true;
    }
    
    // Количество — O(1)
    public function count(): int
    {
        return count($this->tasks);
    }
}

// Использование
$todo = new TodoList();
$todo->add("Купить хлеб");
$todo->add("Позвонить маме");
$todo->add("Прочитать книгу");

echo "Задач: " . $todo->count() . "\n";
$todo->showAll();

echo "\nВыполняем задачу 2:\n";
$todo->complete(2);
$todo->showAll();
```

**Вывод:**
```
Задач: 3
1. Купить хлеб
2. Позвонить маме
3. Прочитать книгу

Выполняем задачу 2:
1. Купить хлеб
2. Прочитать книгу
```

## Когда массив — хороший выбор?

```
✅ ХОРОШО использовать массив, когда:
   - Нужен быстрый доступ по индексу
   - Данные добавляются/удаляются в основном в конце
   - Размер известен заранее или редко меняется

❌ ПЛОХО использовать массив, когда:
   - Часто добавляем/удаляем в начале или середине
   - Нужен быстрый поиск по значению (без сортировки)
```

## Визуализация: Что происходит в памяти

```php
<?php

$numbers = [10, 20, 30, 40, 50];
```

```
Память компьютера (упрощённо):

Адрес:  0x100  0x104  0x108  0x112  0x116
        ┌──────┬──────┬──────┬──────┬──────┐
Данные: │  10  │  20  │  30  │  40  │  50  │
        └──────┴──────┴──────┴──────┴──────┘
        [0]    [1]    [2]    [3]    [4]  ← индексы

Каждый элемент занимает 4 байта (для int)
Элементы лежат РЯДОМ друг с другом
Поэтому доступ по индексу — O(1): просто сложить адрес + индекс
```

## Упражнения

### Легко
Создай массив из 5 имён и выведи каждое имя на новой строке с номером.

### Средне
Напиши функцию, которая:
- Принимает массив чисел
- Возвращает новый массив, где каждое число умножено на 2

```php
doubleAll([1, 2, 3]); // [2, 4, 6]
```

### Сложно
Напиши функцию, которая удаляет из массива все чётные числа:

```php
removeEvens([1, 2, 3, 4, 5, 6]); // [1, 3, 5]
```

---

## Решения

### Легко
```php
<?php

$names = ["Аня", "Борис", "Вика", "Глеб", "Даша"];

for ($i = 0; $i < count($names); $i++) {
    echo ($i + 1) . ". " . $names[$i] . "\n";
}
```

### Средне
```php
<?php

function doubleAll(array $numbers): array
{
    $result = [];
    
    foreach ($numbers as $num) {
        $result[] = $num * 2;
    }
    
    return $result;
}

// Или короче:
function doubleAllShort(array $numbers): array
{
    return array_map(fn($n) => $n * 2, $numbers);
}
```

### Сложно
```php
<?php

function removeEvens(array $numbers): array
{
    $result = [];
    
    foreach ($numbers as $num) {
        if ($num % 2 !== 0) {
            $result[] = $num;
        }
    }
    
    return $result;
}

// Или:
function removeEvensShort(array $numbers): array
{
    return array_values(array_filter($numbers, fn($n) => $n % 2 !== 0));
}
```

## Итоги

- **Массив** — это набор элементов с индексами
- Индексы начинаются с **0**
- Доступ по индексу — **O(1)**
- Добавление в конец — **O(1)** амортизированно
- Добавление/удаление в начале — **O(n)**
