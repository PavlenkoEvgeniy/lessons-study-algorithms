# Урок 6: Хеш-таблицы (Hash Tables)

## Зачем этот урок

**Проблема:** У тебя 10,000 друзей в телефоне. Как найти номер Маши?
- Линейный поиск: проверить 10,000 контактов? **O(n) — медленно**
- По алфавиту: "М" — где-то в середине? **O(log n) — лучше, но всё равно искать**

**Решение:** Хеш-таблица. Позволяет искать за **O(1)** — мгновенно!

---

## Что такое хеш-таблица?

**Хеш-таблица** — это структура, которая превращает "ключ" (например, имя) в "адрес" (номер ячейки), где лежит значение.

**Простая аналогия: Почтовые ящики**

```
Обычный поиск:   Проверить все ящики по очереди
Хеш-таблица:     "Маша" → хеш-функция → ящик #47 → сразу открываем!

Хеш-функция:  hash("Маша") = 47
               hash("Петя") = 23
               hash("Аня") = 15
```

---

## Как работает хеш-функция?

**Хеш-функция** — это формула, которая превращает ключ в число.

```php
<?php

/**
 * Простая хеш-функция для строк
 * Складывает коды символов и берёт остаток от деления на размер
 */
function simpleHash(string $key, int $size): int
{
    $hash = 0;
    
    for ($i = 0; $i < strlen($key); $i++) {
        $hash += ord($key[$i]);  // ord() — код символа
    }
    
    return $hash % $size;  // Остаток от деления на размер
}

// Тест
$size = 10;
echo simpleHash("Маша", $size);  // ?
echo simpleHash("Петя", $size);  // ?
echo simpleHash("Аня", $size);   // ?
```

**Принцип:**
1. Берём ключ ("Маша")
2. Хеш-функция выдаёт число (например, 47)
3. Число 47 — это индекс в массиве, где лежит значение

## Реализация на PHP

```php
<?php

/**
 * Простая хеш-таблица
 */
class HashTable
{
    private array $buckets;  // Массив "ведёрок"
    private int $size;       // Размер таблицы
    
    public function __construct(int $size = 16)
    {
        $this->size = $size;
        $this->buckets = array_fill(0, $size, []);
    }
    
    /**
     * Хеш-функция
     */
    private function hash(string $key): int
    {
        $hash = 0;
        for ($i = 0; $i < strlen($key); $i++) {
            $hash = ($hash + ord($key[$i])) % $this->size;
        }
        return $hash;
    }
    
    /**
     * Добавить значение — O(1) в среднем
     */
    public function set(string $key, mixed $value): void
    {
        $index = $this->hash($key);
        
        // Ищем существующую пару с таким ключом
        foreach ($this->buckets[$index] as &$pair) {
            if ($pair['key'] === $key) {
                $pair['value'] = $value;  // Обновляем
                return;
            }
        }
        
        // Не нашли — добавляем новую пару
        $this->buckets[$index][] = ['key' => $key, 'value' => $value];
    }
    
    /**
     * Получить значение — O(1) в среднем
     */
    public function get(string $key): mixed
    {
        $index = $this->hash($key);
        
        // Ищем в этом "ведёрке"
        foreach ($this->buckets[$index] as $pair) {
            if ($pair['key'] === $key) {
                return $pair['value'];
            }
        }
        
        return null;  // Не нашли
    }
    
    /**
     * Проверить существование — O(1)
     */
    public function has(string $key): bool
    {
        return $this->get($key) !== null;
    }
    
    /**
     * Удалить — O(1)
     */
    public function delete(string $key): bool
    {
        $index = $this->hash($key);
        
        foreach ($this->buckets[$index] as $i => $pair) {
            if ($pair['key'] === $key) {
                array_splice($this->buckets[$index], $i, 1);
                return true;
            }
        }
        
        return false;
    }
}
```

## Проблема: Коллизии

**Коллизия** — это когда два разных ключа дают одинаковый хеш.

```
hash("Маша") = 5
hash("Лена") = 5  ← Коллизия! Оба хотят в ячейку 5
```

**Как решаем:** В каждой ячейке может быть НЕ один элемент, а СПИСОК элементов (связный список). Это называется **separate chaining** (раздельная цепочка).

```
Индекс 5 после добавления "Маша" и "Лена":
┌─────────────────────────────────────┐
│  bucket[5]                          │
│                                     │
│  ┌───────────┐  ┌───────────┐      │
│  │ key: Маша │→ │ key: Лена │→ null│
│  │ value:...│  │ value:... │      │
│  └───────────┘  └───────────┘      │
└─────────────────────────────────────┘

При поиске "Лена":
1. hash("Лена") → индекс 5
2. Ищем в списке внутри bucket[5]
3. Нашли "Лена" → вернули значение
```

---

## Сравнение: Массив vs Связный список vs Хеш-таблица

```
┌──────────────────────┬─────────────────┬─────────────────┬─────────────────┐
│ Операция             │ Массив          │ Связный список   │ Хеш-таблица     │
├──────────────────────┼─────────────────┼─────────────────┼─────────────────┤
│ Поиск по ключу       │ O(n)            │ O(n)            │ O(1) ✅         │
│ Добавление           │ O(1)*           │ O(1)            │ O(1) ✅         │
│ Удаление             │ O(n)            │ O(n)            │ O(1) ✅         │
│ Упорядоченность      │ По индексу      │ Нет             │ Нет             │
└──────────────────────┴─────────────────┴─────────────────┴─────────────────┘
* амортизированно
```

## Пример: Телефонная книга

```php
<?php

/**
 * Телефонная книга на хеш-таблице
 */
class PhoneBook
{
    private HashTable $contacts;
    
    public function __construct()
    {
        $this->contacts = new HashTable(100);  // 100 "ячеек"
    }
    
    /**
     * Добавить контакт
     */
    public function add(string $name, string $phone): void
    {
        $this->contacts->set($name, $phone);
        echo "Добавлен: $name\n";
    }
    
    /**
     * Найти номер
     */
    public function find(string $name): ?string
    {
        $phone = $this->contacts->get($name);
        if ($phone) {
            echo "$name: $phone\n";
        } else {
            echo "$name: не найден\n";
        }
        return $phone;
    }
    
    /**
     * Удалить контакт
     */
    public function remove(string $name): void
    {
        if ($this->contacts->delete($name)) {
            echo "Удалён: $name\n";
        } else {
            echo "$name: не был найден\n";
        }
    }
}

// Использование
$book = new PhoneBook();

$book->add("Маша", "+7-999-123-45-67");
$book->add("Петя", "+7-999-987-65-43");
$book->add("Аня", "+7-999-111-22-33");
$book->add("Борис", "+7-999-222-33-44");

echo "\nПоиск:\n";
$book->find("Маша");
$book->find("Коля");  // Не существует

echo "\nУдаление:\n";
$book->remove("Петя");

echo "\nПовторный поиск:\n";
$book->find("Петя");  // Уже удалён
```

**Вывод:**
```
Добавлен: Маша
Добавлен: Петя
Добавлен: Аня
Добавлен: Борис

Поиск:
Маша: +7-999-123-45-67
Коля: не найден

Удаление:
Удалён: Петя

Повторный поиск:
Петя: не найден
```

## Встроенная хеш-таблица в PHP

В PHP массив (`array`) — это уже хеш-таблица!

```php
<?php

// PHP массив — это хеш-таблица!
$phoneBook = [
    "Маша" => "+7-999-123-45-67",
    "Петя" => "+7-999-987-65-43",
    "Аня" => "+7-999-111-22-33",
];

// Добавление — O(1)
$phoneBook["Борис"] = "+7-999-222-33-44";

// Поиск — O(1)
echo $phoneBook["Маша"];  // "+7-999-123-45-67"

// Проверка — O(1)
isset($phoneBook["Маша"]);  // true

// Удаление — O(1)
unset($phoneBook["Петя"]);
```

**Примечание:** PHP-arrays поддерживают и упорядоченные числовые ключи тоже, поэтому они "гибридные".

---

## Когда использовать хеш-таблицу?

```
✅ ХОРОШО:
   - Нужен быстрый поиск по ключу
   - Ключи — строки или числа
   - Не важна сортировка

❌ ПЛОХО:
   - Нужна сортировка элементов
   - Ключи — объекты без хорошей хеш-функции
   - Данные очень маленькие (массив будет быстрее)
```

## Упражнения

### Легко
Создай хеш-таблицу для хранения возраста друзей и найди возраст конкретного друга.

### Средне
Напиши функцию, которая находит первое неповторяющееся слово в строке:
```php
firstUnique("привет мир привет"); // "мир"
firstUnique("а а б б а");          // "б"
```

### Сложно
Реализуй хеш-таблицу с отдельными цепочками (separate chaining) для обработки коллизий.

## Решения

### Легко
```php
<?php

$ages = [
    "Маша" => 25,
    "Петя" => 30,
    "Аня" => 28,
    "Борис" => 35,
];

echo $ages["Маша"];  // 25

// Или через цикл
foreach ($ages as $name => $age) {
    echo "$name: $age лет\n";
}
```

### Средне
```php
<?php

function firstUnique(string $text): ?string
{
    $words = explode(" ", $text);
    $counts = [];  // Хеш-таблица: слово → количество
    
    // Считаем все слова
    foreach ($words as $word) {
        if (!isset($counts[$word])) {
            $counts[$word] = 0;
        }
        $counts[$word]++;
    }
    
    // Ищем первое уникальное
    foreach ($words as $word) {
        if ($counts[$word] === 1) {
            return $word;
        }
    }
    
    return null;
}

echo firstUnique("привет мир привет"); // "мир"
```

### Сложно
```php
<?php

/**
 * Хеш-таблица с отдельными цепочками
 */
class HashTableWithChaining
{
    private array $buckets;  // Массив связных списков
    private int $size;
    private int $count = 0;
    
    public function __construct(int $size = 16)
    {
        $this->size = $size;
        $this->buckets = array_fill(0, $size, null);
    }
    
    private function hash(string $key): int
    {
        $hash = 0;
        for ($i = 0; $i < strlen($key); $i++) {
            $hash = ($hash + ord($key[$i])) % $this->size;
        }
        return $hash;
    }
    
    public function set(string $key, mixed $value): void
    {
        $index = $this->hash($key);
        
        // Ищем существующий узел
        $node = $this->buckets[$index];
        while ($node !== null) {
            if ($node['key'] === $key) {
                $node['value'] = $value;
                return;
            }
            $node = $node['next'];
        }
        
        // Добавляем в начало списка
        $this->buckets[$index] = [
            'key' => $key,
            'value' => $value,
            'next' => $this->buckets[$index]
        ];
        $this->count++;
    }
    
    public function get(string $key): mixed
    {
        $index = $this->hash($key);
        $node = $this->buckets[$index];
        
        while ($node !== null) {
            if ($node['key'] === $key) {
                return $node['value'];
            }
            $node = $node['next'];
        }
        
        return null;
    }
}
```

## Итоги

- **Хеш-таблица** — быстрый поиск за O(1)
- **Хеш-функция** — превращает ключ в индекс
- **Коллизии** — решаются через цепочки (separate chaining)
- В PHP массив — это уже хеш-таблица!
