# План изучения алгоритмов и структур данных на PHP

## Главный принцип
> "Лучше 10 уроков, которые понятны, чем 20, которые пугают"

## Структура: 12 уроков

### Секция 1. Введение (2 урока)
1. **[Что такое алгоритм](lessons/01-introduction.md)** — простые примеры из жизни, что такое инструкция
2. **[Как измерить скорость](lessons/02-big-o-notation.md)** — Big O, зачем это нужно, только главное (O(1), O(n), O(log n), O(n²))

### Секция 2. Простые структуры (3 урока)
3. **[Массивы](lessons/03-arrays.md)** — что это и зачем, доступ по индексу, простое объяснение
4. **[Связные списки](lessons/04-linked-lists.md)** — следующий элемент, плюсы/минусы vs массива
5. **[Стеки и очереди](lessons/05-stacks-queues.md)** — LIFO и FIFO, где применяются (вместе, потому что простые)

### Секция 3. Хеш-таблицы (1 урок)
6. **[Хеш-таблицы](lessons/06-hash-tables.md)** — быстрый поиск, хеш-функция, коллизии

### Секция 4. Деревья (2 урока)
7. **[Деревья и обходы](lessons/07-trees-and-traversals.md)** — что такое дерево, BFS, DFS (просто!)
8. **[Бинарное дерево поиска](lessons/08-binary-search-trees.md)** — поиск, вставка (без удаления для простоты)

### Секция 5. Сортировка (1 урок)
9. **[Алгоритмы сортировки](lessons/09-sorting-algorithms.md)** — пузырёк, выбор, вставка, быстрая (4 штуки, понятно)

### Секция 6. Графы (2 урока)
10. **[Графы: основы](lessons/10-graphs-basics.md)** — что такое граф, представление, BFS, DFS
11. **[Кратчайший путь](lessons/11-shortest-path-dijkstra.md)** — алгоритм Дейкстры (простое объяснение)

### Секция 7. Продвинутые темы (1 урок)
12. **[Динамическое программирование](lessons/12-dynamic-programming.md)** — запоминание результатов, Fibonacci, задача о рюкзаке

---

## Формат каждого урока

Каждый урок содержит:
1. **Зачем это нужно** — одна картинка из жизни, проблема
2. **Объяснение** — пошагово, с аналогиями
3. **Код на PHP** — минимальный, понятный
4. **Визуализация** — ASCII-картинка что происходит
5. **Когда использовать** — практическое применение
6. **Упражнения** — 3 уровня: легко/средне/сложно

---

## Что НЕ вошло в план (для продвинутых)

Эти темы — для отдельного курса:
- AVL-деревья, красно-чёрные деревья
- Жадные алгоритмы (кроме краткого упоминания)
- Беллман-Форд, Флойд-Уоршалл
- Топологическая сортировка
- Рекурсия глубокая
- Backtracking

**Принцип:** базовый курс должен дать фундамент. Продвинутые темы — потом, когда foundation есть.
