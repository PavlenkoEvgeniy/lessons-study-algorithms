# План изучения алгоритмов и структур данных на PHP

## Главный принцип
> "Лучше 10 уроков, которые понятны, чем 20, которые пугают"

## Структура: 12 уроков

### Фаза 1: Введение (2 урока)
1. **Что такое алгоритм** — простые примеры из жизни, что такое инструкция
2. **Как измерить скорость** — Big O, зачем это нужно, только главное (O(1), O(n), O(log n), O(n²))

### Фаза 2: Простые структуры (3 урока)
3. **Массивы** — что это и зачем, доступ по индексу, простое объяснение
4. **Связные списки** — следующий элемент, плюсы/минусы vs массива
5. **Стеки и очереди** — LIFO и FIFO, где применяются (вместе, потому что простые)

### Фаза 3: Хеш-таблицы (1 урок)
6. **Хеш-таблицы** — быстрый поиск, хеш-функция, коллизии

### Фаза 4: Деревья (2 урока)
7. **Деревья и обходы** — что такое дерево, BFS, DFS (просто!)
8. **Бинарное дерево поиска** — поиск, вставка (без удаления для простоты)

### Фаза 5: Сортировка (1 урок)
9. **Алгоритмы сортировки** — пузырёк, выбор, вставка, быстрая (4 штуки, понятно)

### Фаза 6: Графы (2 урока)
10. **Графы: основы** — что такое граф, представление, BFS, DFS
11. **Кратчайший путь** — алгоритм Дейкстры (простое объяснение)

### Фаза 7: Продвинутые темы (1 урок)
12. **Динамическое программирование** — запоминание результатов, Fibonacci, задача о рюкзаке

---

## Формат каждого урока

Каждый урок содержит:
1. **Зачем это нужно** — одна картинка из жизни, проблема
2. **Объяснение** — пошагово, с аналогиями
3. **Код на PHP** — минимальный, понятный
4. **Визуализация** — ASCII-картинка что происходит
5. **Когда использовать** — практическое применение
6. **Упражнения** — 3 уровня: легко/средне/сложно

---

## Что НЕ вошло в план (для продвинутых)

Эти темы — для отдельного курса:
- AVL-деревья, красно-чёрные деревья
- Жадные алгоритмы (кроме краткого упоминания)
- Беллман-Форд, Флойд-Уоршалл
- Топологическая сортировка
- Рекурсия глубокая
- Backtracking

**Принцип:** базовый курс должен дать фундамент. Продвинутые темы — потом, когда foundation есть.
